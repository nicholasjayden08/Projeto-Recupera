# Projeto Recupera

A student management system built in Java with dynamic grade registration and individual performance tracking.

## Features

- ✅ Add students to the system
- ✅ Add grades dynamically (one at a time)
- ✅ View individual student grades and performance
- ✅ List all registered students
- ✅ Search for students by name
- ✅ Remove students from the system
- ✅ Calculate class average
- ✅ Track student progression status (passed/failed)

## Project Structure
src/Praticando/Pessoal/
├── ProjetoRecuperaAluno.java        - Student model with grade management
├── ProjetoRecuperaSistema.java      - System that manages all students
└── ProjetoRecuperaMain.java         - Interactive menu interface

## How to Run

1. Open the project in IntelliJ IDEA
2. Run the `ProjetoRecuperaMain` class
3. Follow the menu options to manage students

## Technologies Used

- Java 11+
- Object-Oriented Programming (OOP)
- ArrayList for dynamic data storage
- Exception handling
